# 一括調べものツール

複数の生成AIサービス（Gemini, ChatGPT, Claude, Perplexity）に対して、ひとつの入力画面から同時に質問を送信し、回答を並行して確認し、さらに特定のAIを使って各回答を比較・要約（まとめる）できるWindows用デスクトップツールです。

## ダウンロード・導入方法

### A. ZIPファイルでダウンロードする場合（Git が不要・初心者向け）

1. このページ右上の **`Code`** ボタンをクリックします。
2. **`Download ZIP`** を選択してZIPファイルをダウンロードします。
3. ダウンロードしたZIPを右クリック ＞「すべて展開...」で任意のフォルダに解凍します。
4. 解凍したフォルダを開き、「使い方・セットアップ」の手順へ進んでください。

### B. Git でクローンする場合

```bash
git clone https://github.com/katojishiKun/Generative-AI-Bulk-Search.git
cd Generative-AI-Bulk-Search
```

## 使い方・セットアップ

### 1. 初回のみ：ログインセットアップ（`生成AI初回設定.bat`）

> **これは初回だけ必要な手順です。** 2回目以降は不要です。

1. フォルダ内の **`生成AI初回設定.bat`** をダブルクリックして実行します。
2. ツール専用の Google Chrome ウィンドウが開きます。  
   *(これは自動化なしの「通常のChromeとして」起動されるため、Cloudflare等のセキュリティチェックも正常に通過できます)*
3. 開いた Chrome で以下の各サービスにアクセスし、**手動でログイン**します。

   | サービス | URL |
   |---|---|
   | Gemini | https://gemini.google.com/ |
   | ChatGPT | https://chatgpt.com/ |
   | Claude | https://claude.ai/ |
   | Perplexity | https://www.perplexity.ai/ |

4. 全サービスへのログインが完了したら、Chrome を**完全に閉じます**。

> [!IMPORTANT]  
> **Chrome を完全に閉じてから、次の手順に進んでください。**  
> 同じ専用プロファイルを2つのChromeで同時に使うことはできません。バックグラウンドにChromeのプロセスが残ったままの状態で `生成AI一括検索.bat` を実行すると、プロファイルがロックされてエラーになります。タスクバーにChromeのアイコンが残っていないことを確認してください。

### 2. 通常使用時（`生成AI一括検索.bat`）

1. **`生成AI一括検索.bat`** をダブルクリックして実行します。
2. 初回起動時は必要なパッケージ（Playwright 等）が自動的にダウンロードされるため、しばらく待ちます。
3. ツール専用の Google Chrome が起動し、各 AI サービスのタブと入力ページが自動的に開きます。
4. **一括検索する**: 左側の「送信先」を選択し、テキストエリアに質問を入力して「送信」ボタンを押してください。
5. **回答をまとめる**: 各AIの回答が揃った後、右側の「まとめる」エリアでまとめ先とするAIを選択し、指示プロンプトを入力して「送信」ボタンを押すと、各AIの回答内容を比較・要約したまとめ結果が最新履歴として最上部に追加されます。

> **補足**: 本ツールは普段使いの Google Chrome とは**完全に独立した専用プロファイル**（`chrome-profile/` フォルダ）を使用します。  
> 普段の Chrome と同時に起動でき、互いのログイン状態に影響しません。
>
> **ログイン情報の引き継ぎについて**: `生成AI初回設定.bat` でログインした情報（Cookie・セッション情報）は、専用プロファイル（`chrome-profile/` フォルダ）内に自動的に保存されます。`生成AI一括検索.bat` を実行すると同じ専用プロファイルが読み込まれるため、改めてログインすることなく、各 AI サービスがログイン済みの状態で開きます。
>
> **セッションが切れた場合**: 各AIサービス側でセッションが切れてログアウトされた場合は、再度 `生成AI初回設定.bat` を実行して手動ログインし直してください。その後は再び `生成AI一括検索.bat` だけで利用できます。

## よくある質問（トラブルシューティング）

**Q. `生成AI一括検索.bat` を実行するとすぐに黒い画面が閉じてしまう**  
A. Node.js がインストールされていない可能性があります。[Node.js 公式サイト](https://nodejs.org/ja/) からインストールしてください。インストール後はPCを再起動してから再度実行してください。

**Q. Chrome が起動するが、各AIサービスでログアウト状態になっている**  
A. セッションが切れている状態です。一度 Chrome を閉じてから `生成AI初回設定.bat` を実行し、手動ログインし直してください。

**Q. 「プロファイルがロックされています」のようなエラーが出る**  
A. バックグラウンドで Chrome が起動したままになっています。タスクマネージャーで Chrome プロセスをすべて終了してから再度 `生成AI一括検索.bat` を実行してください。

**Q. 回答の取得がタイムアウトする**  
A. 各AIサービス側の応答が遅い場合や、UIの仕様変更があった場合に発生することがあります。時間を置いて再度お試しください。

**Q. Windows のセキュリティ警告（SmartScreen）が出る**  
A. ダウンロードした `.bat` ファイルを初めて実行する際に出ることがあります。「詳細情報」＞「実行」をクリックすれば実行できます。ソースコードはすべて公開されているため、安全を確認したうえでご利用ください。

**Q. ロボット認証（CAPTCHAや「人間であることを確認します」など）が表示されて回答が取得できない**  
A. 自動操作によるアクセス判定等により、セキュリティチェックに引っかかる場合があります。この場合は、一度ツールを完全に閉じてから **`生成AI初回設定.bat`** を実行してください。手動操作用のブラウザが開くので、そこで該当するAIサービスを開き、ロボット認証を人間の手で通過させてください。認証通過後にブラウザを閉じ、再度 `生成AI一括検索.bat` を実行すれば解消されます。
