# 一括調べものツール

複数の生成AIサービス（Gemini, ChatGPT, Claude, Perplexity）に対して、ひとつの入力画面から同時に質問を送信し、回答を並行して確認できるWindows用デスクトップツールです。

## 使い方・セットアップ

### 1. 初回のみ：ログインセットアップ（`setup_login.bat`）

> **これは初回だけ必要な手順です。** 2回目以降は不要です。

1. フォルダ内の **`setup_login.bat`** をダブルクリックして実行します。
2. ツール専用の Google Chrome ウィンドウが開きます。  
   *(これは自動化なしの「通常のChromeとして」起動されるため、Cloudflare等のセキュリティチェックも正常に通過できます)*
3. 開いた Chrome で以下の各サービスにアクセスし、**手動でログイン**します。

   | サービス | URL |
   |---|---|
   | Gemini | https://gemini.google.com/ |
   | ChatGPT | https://chatgpt.com/ |
   | Claude | https://claude.ai/ |
   | Perplexity | https://www.perplexity.ai/ |

4. 全サービスへのログインが完了したら、Chrome を**完全に閉じます**。

> [!IMPORTANT]  
> **Chrome を完全に閉じてから、次の手順に進んでください。**  
> 同じ専用プロファイルを2つのChromeで同時に使うことはできません。バックグラウンドにChromeのプロセスが残ったままの状態で `run_chrome.bat` を実行すると、プロファイルがロックされてエラーになります。タスクバーにChromeのアイコンが残っていないことを確認してください。

### 2. 通常使用時（`run_chrome.bat`）

1. **`run_chrome.bat`** をダブルクリックして実行します。
2. 初回起動時は必要なパッケージ（Playwright 等）が自動的にダウンロードされるため、しばらく待ちます。
3. ツール専用の Google Chrome が起動し、各 AI サービスのタブと入力ページが自動的に開きます。
4. 送信先を選択し、テキストエリアに質問を入力して「送信」ボタンを押してください。

> **補足**: 本ツールは普段使いの Google Chrome とは**完全に独立した専用プロファイル**（`chrome-profile/` フォルダ）を使用します。  
> 普段の Chrome と同時に起動でき、互いのログイン状態に影響しません。
>
> **ログイン情報の引き継ぎについて**: `setup_login.bat` でログインした情報（Cookie・セッション情報）は、専用プロファイル（`chrome-profile/` フォルダ）内に自動的に保存されます。`run_chrome.bat` を実行すると同じ専用プロファイルが読み込まれるため、改めてログインすることなく、各 AI サービスがログイン済みの状態で開きます。
>
> **セッションが切れた場合**: 各AIサービス側でセッションが切れてログアウトされた場合は、再度 `setup_login.bat` を実行して手動ログインし直してください。その後は再び `run_chrome.bat` だけで利用できます。
